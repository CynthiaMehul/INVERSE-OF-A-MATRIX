## Algorithm:
### Step1 : 
Import numpy as np
### Step 2: 
Assign matrix to any variable
### Step 3: 
Using the np.linalg.matrix_rank(), we can find the rank of the given matrix.
### Step 4: 
Print the derived answer

## Program:
